## 📌 About the Project

This project is a simple C++ console application that demonstrates object-oriented programming using inheritance.

It models two types of users:
- Operario (Worker)
- Manager

---

## 🚀 Features

- 🧱 Base class: Usuario
- 👷 Derived class: Operario
- 👨‍💼 Derived class: Manager
- 📧 User data handling
- 🖥️ Console output

---

## 🛠️ Technologies

- C++
- Standard Library (`iostream`, `string`)

---

## 💻 How it works

1. Create user objects
2. Assign user data
3. Display formatted output

---

## 🎯 Learning Goals

- Understand inheritance in C++
- Practice class design
- Improve OOP thinking
- Learn encapsulation basics
